Solar system generator (3D)

Con el fin de utilizar a profundidad los conceptos aprendidos en la asignatura, se decidió crear una simulación a escala de nuestro sistema solar por medio de la herramienta Processing 5.

Como primer acercamiento al problema, se realizó una investigación exhaustiva acerca de tamaños, distancias y órbitas de cada elemento del sistema solar, así como el apartado de iluminación y satélites.

Añadir textura a la esfera y rotarla sobre un eje

Las texturas utilizadas se encuentran en http://planetpixelemporium.com/planets.html

                    show() {
                        push();
                        noStroke();
                        let v2 = createVector(1, 0, 1);
                        let p = this.v.cross(v2);
                        
                        if (p.x != 0 || p.y != 0 || p.z != 0) {
                          rotate(this.angle, p);
                        }
                        stroke(255);
                    
                        translate(this.v.x, this.v.y, this.v.z);
                        noStroke();
                        fill(255);
                        
                        texture(this.texture);
                        sphere(this.radius);
                        
                        if (this.planetas != null) {
                          for (let i = 0; i < this.planetas.length; i++) {
                            this.planetas[i].show();
                          }
                        }
                        pop();
                      }

Función que cambia el ángulo del planeta y sus satélites

orbit() {
    this.angle = this.angle + this.orbitspeed;
    if (this.planetas != null) {
        for (let i = 0; i < this.planetas.length; i++) {
            this.planetas[i].orbit();
        }
    }
}
